# Release v0.4.0 — Skills / Tools Tab
**Date:** 2025-07-14
**Type:** Minor (new feature, backwards-compatible)
**Scope:** Frontend only — `apps/dashboard`

## Highlights

Adds a tenth tab, **"Skills"**, to each live agent's detail page (`/agents/:id`).
Operators can now inspect exactly which tools, skills, MCP servers, and identity
properties are active for any running agent — without leaving the dashboard.

No backend changes, no database migrations, no new API endpoints.

## What's New

### `AgentSkillsTab` component
A new React component that fetches from the existing `GET /api/v2/agents/:id/manifest`
endpoint and renders four panels: Tools, Skills, MCP Servers, Character & Identity.

### `AgentDashboard.tsx` changes
- Tab bar expanded: `grid-cols-9` → `grid-cols-10`
- New `TabsTrigger value="skills"` with Wrench icon
- New `TabsContent value="skills"` rendering `<AgentSkillsTab />`

## Breaking Changes
None.

## Migration Guide
None required.

## Rollback Plan

```bash
git revert HEAD --no-edit
git push origin main
```

## Verification Steps (post-deploy)

1. Navigate to any live agent → confirm 10 tabs visible.
2. Click **Skills** tab → verify Tools, Skills, MCP Servers, and Character panels render.
3. Check browser Network tab → single call to `/api/v2/agents/:id/manifest`.
4. Confirm no console errors.
